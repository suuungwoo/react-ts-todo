# React TypeScript Todo
TypeScript, styled-componentsの練習で作ったTodoアプリです。

# Setup

```
$ git clone https://github.com/suuungwoo/react-todos.git
$ yarn
$ yarn start
```

# Usage
タスクを追加したり完了にしたりフィルタリングしたり
